STREAMS ONLINE 4.1.3 Score Highlight Safe

기준:
- 정상 접속 확인된 4.1.2 SAFE 기반입니다.

추가:
- 점수에 포함된 오름차순 칸: 초록색 표시
- 점수에 포함되지 못한 칸: 빨간색 표시

안전 적용 방식:
- 기존 simpleBoard / scoreBoard 함수는 건드리지 않았습니다.
- 기존 renderBoard 결과 위에 색상 클래스만 추가합니다.
- 하이라이트 분석 중 오류가 나도 기존 화면은 계속 표시됩니다.

기존 기능 유지:
- 재입장
- 실시간 순위
- TOP3 비교
- 학생 보드 보기
- 방코드 복사
