STREAMS ONLINE 3.1 Beta Realtime Score Fix

수정 사항:
1. 점수 계산 수정
   - 감소 지점에서 오름차순 묶음을 끊고 각 묶음 점수를 합산합니다.
   - 예: 12, 12, 23, 14, 18
     → 12,12,23 = 3칸 = 3점
     → 14,18 = 2칸 = 1점
     → 총 4점
   - 빈칸으로 끊어진 묶음도 각각 계산합니다.

2. 실시간 반영 안정화
   - Firebase Streaming + 0.7초 폴링 fallback을 함께 사용합니다.
   - 교사용 학생목록 입력/미입력 반영 안정화
   - 전자칠판 순위 실시간 반영 안정화
