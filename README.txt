STREAMS ONLINE 3.3.2 Client Online

추가/수정:
- clientId + online 방식으로 같은 이름 중복 접속 차단
- 같은 브라우저/기기에서 새로고침 또는 재입장하면 이어서 진행
- 다른 기기에서 같은 이름이 이미 online=true이면 입장 차단
- 안내 문구: 이미 같은 이름으로 다른 기기에서 접속 중입니다. 다른 이름을 입력해주세요.
- heartbeat 시간 기준 차단은 사용하지 않고, online/clientId 중심으로 동작
- 기존 3.3.1 기능 유지: 이름 입력 화면 먼저 표시, 같은 이름 재입장 이어서 진행, 실시간 점수/순위 유지
